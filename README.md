## 1. Introduction
Our goal was to find the similar traits of Shakespeare's works to find categories we can assign these to. 

### 🎯 Our Objectives:
- Set corpus methodology
- Set sample data
- Set classification types
- Analyze the sentiment graph and the blob graph

### 💡 What We Already Knew:
- Shakespeare's works contain rich emotional and relational patterns; sentiment analysis can measure emotional flow in text.
- Shakespeare's works can be roughly divided into tragedy and comedy, showing a clear distinction.
- Each Shakespeare type must include viewer-focused labels for streaming audiences.

This led us to analyze Shakespeare’s Great 4 Tragedies and major comedies to get a grip on what we are dealing with.

---

## 2. Methodology
We used **Gephi lite** and **Voyant Tools** to turn emotional flow into something we can measure. We used overall sentiment to determine the tone, Gephi lite tool to determine how close the protagonist is to the heroine, and a pleasure & pain graph to measure how dramatic the work is.

### 📐 Classification Standards (6 Attributes):
1. **Main character centric** vs **Character sharing spotlight**
2. **Romantic-centric** vs **Not romantic-centric** (히로인과의 선이 얼마나 굵은가)
3. **소수극** vs **다수극**
4. **Happy ending** vs **Sad ending** vs **Hopeful ending** (끝이 올라감 vs 끝이 내려감 vs 끝이 내려가다 올라감)
5. **Dramatic** vs **Slow burn** (Sentimental 굴곡에 의해)
6. **Big feelings** vs **Serene feelings** (Pleasure과 Pain이 교차하는 넓이가 얼마나 큰지에 따라 분류)

### 🏷️ Viewer-Focused Streaming Labels:
1. Character-Driven Lead Stories / Ensemble Character Stories
2. Romance-Forward Series / Romance-Light Stories
3. Intimate Cast Dramas / Epic Ensemble Sagas
4. Feel-Good Endings / Tragic Descents / Bittersweet Recoveries
5. High-Intensity Dramas / Slow-Burn Journeys
6. Emotionally Explosive Stories / Calm Atmospheric Stories

> **Limitation:** Because we didn’t know how to precisely measure how “thick” the line should be with the romantic interest, we had to lean into intuition. We discussed how much thickness should be considered ‘enough’, but we wish we could have done a better quantitative job on it.

## 5. Reflection and Conclusion
The 9 categories would cover most of the genres that are in our lives, as it has affected our life tremendously in the past. The reason why it is so significant is because there are a lot of combinations of these 6 traits, yet there are patterns that are frequent. For example, *Hamlet* and *Macbeth* share a lot of characteristics with recent media, like *Game of Thrones*.

### ⚠️ Limitations:
There might be another feature that these things might have. For example, we didn’t utilize horror elements much. What if we used something that indicates horror? We didn’t measure horror because we couldn’t at the time, so without it, it might be incomplete. Though, we shaped a ground to stand on, and with this structure, we can always add another dataset and analyze it to add to the characteristics.
